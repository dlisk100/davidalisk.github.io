## About This Visualization

This interactive visualization shows film permit data across New York City. The map displays permit density by zip code, with controls to filter by time period and permit type.

### How It Works

The visualization is built using:

- Leaflet.js for the interactive map
- Custom JavaScript for data handling and controls
- GeoJSON for zip code boundaries and permit data
